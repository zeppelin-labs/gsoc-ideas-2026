# ZestAPI - Google Summer of Code 2026 Project Ideas

Welcome to ZestAPI! We're excited to have you contribute to making ZestAPI the best Python web framework for building modern REST APIs. This document outlines project ideas for GSoC 2026 contributors.

## About ZestAPI

ZestAPI is a modern, production-ready Python web framework that combines the best features of Flask and FastAPI with zero-configuration auto-discovery, built-in security, and enterprise-grade features. Built on top of Starlette and ASGI, it provides high performance with minimal boilerplate.

**Tech Stack**: Python 3.10+, Starlette, ASGI, orjson, JWT, Pydantic  
**Repository**: https://github.com/madnansultandotme/zestapi-python

## How to Apply

### Application Process

1. **Get Familiar with ZestAPI**
   - Install ZestAPI: `pip install zestapi`
   - Run examples: `cd examples/basic-api && python main.py`
   - Read documentation in `docs/` directory
   - Run tests: `pytest tests/`

2. **Choose a Project**
   - Pick a project that matches your skills and interests
   - Review the project goals and expected outcomes
   - Check the "Getting Started" section

3. **Make a Contribution**
   - Fix a bug or add a small feature
   - Improve documentation
   - Add tests
   - Submit a pull request

4. **Write Your Proposal**
   - Follow GSoC proposal guidelines
   - Include timeline with milestones
   - Show understanding of the project
   - Demonstrate relevant skills
   - Include your contribution history

### Proposal Template

See [PROPOSAL_TEMPLATE.md](./gsoc/PROPOSAL_TEMPLATE.md) for the complete proposal template.

## Evaluation Criteria

We'll evaluate proposals based on:

1. **Technical Understanding** (30%)
   - Understanding of ZestAPI architecture
   - Knowledge of relevant technologies
   - Quality of technical approach

2. **Proposal Quality** (25%)
   - Clear goals and deliverables
   - Realistic timeline
   - Well-structured plan

3. **Prior Contributions** (25%)
   - Pull requests to ZestAPI
   - Code quality
   - Community engagement

4. **Communication** (20%)
   - Clarity of writing
   - Responsiveness
   - Collaboration skills
